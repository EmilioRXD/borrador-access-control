Integración del Loader NFC

Esta página muestra cómo integrar la animación de carga NFC en el sistema de control de acceso.

Demostración

Cómo implementarlo

Para implementar el loader en la página principal, sigue estos pasos:

1. Incluye el script del loader

<script src="loader/nfc-loader.js"></script>

2. Muestra el loader al cargar la página

<script>
document.addEventListener('DOMContentLoaded', function() {
    // Oculta el contenido principal
    const mainContent = document.getElementById('mainContent');
    mainContent.style.opacity = '0';
    
    // Muestra el loader
    const loader = showNfcLoader({
        onComplete: function() {
            // Muestra el contenido principal cuando el loader termina
            mainContent.classList.add('visible');
        }
    });
});
</script>

3. Opciones de configuración

El loader acepta las siguientes opciones:

  • container: El elemento donde se mostrará el loader (por defecto: document.body)
  • autoHide: Si el loader debe ocultarse automáticamente (por defecto: true)
  • loadTime: Tiempo de carga en milisegundos (por defecto: 5000)
  • onComplete: Función a ejecutar cuando el loader termine

Personalización

Puedes personalizar la apariencia del loader modificando el archivo nfc-loader.css.
